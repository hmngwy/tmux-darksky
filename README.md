### Installation

Requires: `python3`

Add your location and Dark Sky API key:

```
set -g @darksky_lon_lat "14.5833,120.9667"
set -g @darksky_api_key "0a6a8952082439b11cdbcXXXXXXXXX"
```

Add the plugin via tpm:

```
set -g @plugin 'hmngwy/tmux-darksky'
```
